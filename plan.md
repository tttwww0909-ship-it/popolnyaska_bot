## Лист "Статистика" (авто-обновление)
Автоматически обновляется при каждом создании заказа и смене статуса (debounce 60 сек):
- **ЗАКАЗЫ** — всего, оплаченных, выполненных, отменённых, в обработке
- **ФИНАНСЫ** — общая выручка, средний чек, конверсия
- **СЕГОДНЯ** — заказы и выручка за текущий день
- **ПО МЕСЯЦАМ** — заказы, выручка, клиенты, конверсия по каждому месяцу
- **ПО РЕГИОНАМ** — заказы, выручка, клиенты, конверсия по каждому региону
- **СПОСОБЫ ОПЛАТЫ** — распределение по ЮMoney / OZON банк / Crypto

### VIP-заказы (> VIP_THRESHOLD ₽, по умолчанию 8 500)
- Принудительная крипто-оплата со скидкой VIP_DISCOUNT (по умолчанию 2%)
- Бонусные баллы стакаются с VIP-скидкой (сначала −VIP_DISCOUNT%, потом баллы)
- Кнопка отказа → отмена заказа
- Все пороги и проценты вынесены в config.py (VIP_THRESHOLD, VIP_DISCOUNT)

### Реферальная система
| Параметр | Значение |
|----------|----------|
| Мин. чек для % бонуса | 300 ₽ |
| Фикс. бонус (чек < 300₽) | 10 баллов |
| Макс. оплата баллами | 50% от заказа |
| Срок годности баллов | 6 месяцев (FIFO-списание) |
| Предупреждение о сгорании | За 7 дней |
| Доступ к реферальной программе | После 1 завершённого заказа |

| Комиссия | % партнёру | % скидка рефералу (1-й заказ) |
|----------|-----------|-------------------------------|
| 20% | 3% | 2% |
| 15% | 3% | 2% |
| 12% | 2% | 2% |
| 11% | 1.5% | 1% |

### CryptoPay
- Автоматические инвойсы через @CryptoBot
- Webhook на порту 8443 с идемпотентностью (таблица payments, UNIQUE payment_id)
- Fallback на ручной перевод (Bybit UID / Telegram Wallet TRC20)
