## Лист "Статистика" (авто-обновление)
Автоматически обновляется при каждом создании заказа и смене статуса:
- **ЗАКАЗЫ** — всего, оплаченных, выполненных, отменённых, в обработке
- **ФИНАНСЫ** — общая выручка, средний чек, макс/мин заказ
- **СЕГОДНЯ** — заказы и выручка за текущий день
- **ПО МЕСЯЦАМ** — заказы, выручка, клиенты, конверсия по каждому месяцу
- **ПО РЕГИОНАМ** — заказы, выручка, клиенты, конверсия по каждому региону
- **СПОСОБЫ ОПЛАТЫ** — распределение по ЮMoney / OZON банк / Crypto

## Реализованный функционал

### ✅ Основной бот
- [x] Команда /start — приветствие + главное меню
- [x] Кнопка "Пополнить Apple ID" — выбор региона
- [x] 5 регионов: 🇺🇸 США, 🇦🇪 ОАЭ, 🇹🇷 Турция, 🇰🇿 Казахстан, 🇸🇦 Саудовская Аравия
- [x] Казахстан: тарифы 5 000 / 10 000 / 15 000 KZT + кастомная сумма 2 000–45 000 KZT (пополнение по почте Apple ID)
- [x] Gift Card регионы (US, AE, TR, SA): тарифы в валюте региона, выдача кода через бот
- [x] Зелёные яблоки (🍏) во всех кнопках и сообщениях
- [x] Расчёт RUB: стоимость в USDT × курс USDT × комиссия региона
- [x] Комиссия по регионам: KZ 15%, US 15%, AE 15%, SA 15%, TR 10%
- [x] Заказы >8 500 ₽ — только крипто-оплата
- [x] Генерация ордера (ORD-XXXX)
- [x] Сохранение заказа в Google Sheets + SQLite
- [x] Антиспам (лимит заказов)

### ✅ Способы оплаты (порядок: ЮMoney → OZON → Крипто)
- [x] ЮMoney (перевод на кошелёк 4100119061571496)
- [x] OZON банк (по ссылке)
- [x] Криптовалюта — 3 способа:
  - Bybit перевод по UID: 364399004
  - Bybit адрес USDT BSC/BEP20
  - Телеграм кошелёк USDT TRC20
- [x] Кнопка "Я оплатил" → запрос скриншота с полной информацией о заказе (регион, тариф, сумма, способ оплаты)
- [x] Скриншот оплаты пересылается админу — заявка уходит только после получения скриншота

### ✅ Админ-панель (/admin)
- [x] Общая статистика (клиенты, заказы, выручка, конверсия)
- [x] Последние 10 заказов
- [x] Изменение статуса заказа (Новый → Оплачен → В обработке → Выполнен / Отменён)
- [x] Уведомление клиента при смене статуса
- [x] Первое уведомление админу — только при получении чека (отсчёт обслуживания)
- [x] KZ: при подтверждении оплаты — запрос почты Apple ID у клиента
- [x] Gift Card: при подтверждении оплаты — кнопка "📤 Отправить код клиенту"

- [x] Кнопка "Пополнение произведено" → статус "Выполнен" (для KZ)
- [x] В уведомлении о полученной почте — полная инфо о заказе (сервис, тариф, сумма)
