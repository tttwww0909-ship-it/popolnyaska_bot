# Popolnyaska Bot — План проекта

## Инфраструктура
- **Локальная папка:** `C:\Users\maksi\OneDrive\Desktop\popolnyaska_bot\`
- **Сервер:** см. .env (Hetzner VPS)
- **Путь на сервере:** `/opt/popolnyaska-bot/`
- **Systemd сервис:** `popolnyaska-bot.service`
- **База данных:** SQLite (orders.db) — основной источник данных; Google Sheets — запись + лист Статистика

## Файлы проекта
- **bot.py** (~2280 строк) — вся логика бота: хендлеры, заказы, оплата, админка, отзывы
- **database.py** (~620 строк, 24 метода) — модуль SQLite
- **requirements.txt** — python-telegram-bot[job-queue]==20.3, gspread, oauth2client, requests, python-dotenv
- **service_account.json** — ключ Google Sheets
- **.env** — токен, ADMIN_ID, реквизиты оплаты

## Столбцы Google Sheets (лист "popolnyaska_bot")
Номер ордера | User_ID | Username | Регион | Тариф | Сумма RUB | Способ оплаты | Дата | Статус

## Лист "Статистика" (авто-обновление)
Автоматически обновляется при каждом создании заказа и смене статуса (debounce 60 сек):
- **ЗАКАЗЫ** — всего, оплаченных, выполненных, отменённых, в обработке
- **ФИНАНСЫ** — общая выручка, средний чек, конверсия
- **СЕГОДНЯ** — заказы и выручка за текущий день
- **ПО МЕСЯЦАМ** — заказы, выручка, клиенты, конверсия по каждому месяцу
- **ПО РЕГИОНАМ** — заказы, выручка, клиенты, конверсия по каждому региону
- **СПОСОБЫ ОПЛАТЫ** — распределение по ЮMoney / OZON банк / Crypto

## Архитектура данных: SQLite vs Google Sheets

| Операция | Источник |
|---------|---------|
| Создание заказа | SQLite + Sheets append_row |
| Обновление статуса | SQLite + Sheets update_cell |
| Запись способа оплаты | Sheets update_cell (кэш sheets_row) |
| Мои заказы (клиент) | SQLite |
| Последние заказы (админ) | SQLite |
| Статистика (админ) | SQLite |
| Управление статусами (админ) | SQLite |
| Информация о заказе (админ) | SQLite |
| Contact manager | SQLite |
| Лист "Статистика" | Sheets get_all_records (фоновый поток) |

### ✅ Основной бот

- [x] Кнопка "Пополнить Apple ID" — выбор региона

- [x] Gift Card регионы (US, AE, TR, SA): тарифы в валюте региона, выдача кода через бот
- [x] Расчёт RUB: стоимость в USDT × курс USDT × комиссия региона
- [x] Комиссия по регионам: KZ 15%, US 15%, AE 15%, SA 15%, TR 10%
- [x] Заказы >8 500 ₽ — только крипто-оплата
- [x] Генерация ордера (ORD-XXXX) с Lock от race condition
- [x] Сохранение заказа в Google Sheets + SQLite
- [x] Антиспам (60 сек кулдаун, макс 3 заказа за 20 мин)
- [x] Кнопка "📎 Отправить другой скриншот" для замены скриншота
